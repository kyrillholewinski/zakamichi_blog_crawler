昨日、11月15日に私が初めて主演を務めた映画「恐怖人形」が公開になりました( ¨̮ )



昨日は舞台挨拶にも行ってきました！

どの回も満席で本当に嬉しかったです(><)

初日からありがとうございました！！




今年の5月下旬頃に撮影をして、もうこの日がやってきました。

とても早いと思うばかりです


主演という事もあり、自分に務まるかどうか、とても不安でした。

でも、撮影当日からスタッフさんや今回共演したキャストの皆様と話していくうちに、緊張でガチガチだった心も、どんどんほぐれていきました

とても居心地のいい空間になってて、終わるのが寂しくなるくらいでした



私にとっては、皆さんがお兄さんやお姉さんのような存在でもあり、素敵な仲間たちです。

すごく助けられました。

これを機に、仲良く話せるようになって良かったです( ¨̮ )




初映画・初主演作品が「恐怖人形」で本当に良かったですし、この作品に携わることが出来て良かったです。


もう皆さんは観てくださいましたか？


私が思うには…の話をしますね


この映画は、怖いのが苦手な人でも観れると思います！人それぞれで、感じる怖さは違ってくると思いますが、怖いだけじゃないです。

笑えるところもあれば、驚くところもあり、そして、感動もあります。


一度観ただけでは気づかなかった点や、気になるところがありましたら、何度も何度も観ていただきたいです。

そしたら、観たときの感想が1回目とは違うかもしれませんよ( ¨̮ )


良かったら、感想など「#恐怖人形」とつけて、SNS等にとうこうしていただきましたら、たくさんの方々の目にとまると思うので、皆さんで共有していきたいです☺️


怖かったら、友達とかと来ると多分見れると思いますよ~

日向坂のメンバーも、本当に怖いのがダメな子は隣の子と手を繋いだまま離さないで、ずーっと見てたメンバーも沢山いたので！




そして、今回主題歌を歌ってくださったThinking Dogsさん。

MVにも出演させていただきました！


事務所の先輩でもあるThinking Dogsさんと、これを機会に共演させていただけて本当にすごく嬉しかったです！

「SPIRAL」大好きで、この映画にとてもピッタリだと思っています。

そして、昨日の舞台挨拶の時に、Thinking Dogsさんからお花も頂きました(><)


とてもお優しい方々ばかりで、コミュニケーションが苦手な私にも沢山話してくださって、おかげで緊張も解けました！

ありがとうございます( ¨̮ )




ここに書ききれなかった事も、全てを含めて、「恐怖人形」は私にとってとても大事な作品ですし大切な思い出になりました。


是非、たくさんたくさん観てください！！


感想も直接聞かせてくださいね☺️☺️


明日は大阪と名古屋に行きますので、待っててね~♪

完売が出たとスタッフさんから聞いて「うそ~！！」となってました( ¨̮ )

さぁ、皆で「恐怖人形」を盛り上げていきましょう☺️
